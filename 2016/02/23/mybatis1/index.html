mybatis 是 java 语言的 orm 框架，使用教程。

这篇主要介绍的是 mybatis 的 Configuration，教程点击这里。

XML 映射配置文件

configuration 节点下的属性：

  • configuration 配置
    • properties 属性
    • settings 设置
    • typeAliases 类型命名
    • typeHandlers 类型处理器
    • objectFactory 对象工厂
    • plugins 插件
    • environments 环境
      • environment 环境变量
        • transactionManager 事务管理器
        • dataSource 数据源
    • databaseIdProvider 数据库厂商标识
    • mappers 映射器

mybatis 的实现当中的 Configuration 基本上就是与这些属性相对应。
